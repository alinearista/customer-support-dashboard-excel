# 📊 Customer Support Performance Dashboard

## Project Overview

This project analyzes 20,000 customer support tickets using Microsoft Excel to evaluate operational efficiency, agent performance, and customer satisfaction.

The objective was to identify key trends, measure support team performance, and generate actionable business recommendations.

---

## Business Problem

A customer support organization wanted to understand:

- Which ticket categories generate the highest volume
- Which priorities require the most resolution time
- Which agents perform best
- Whether resolution time impacts customer satisfaction

## Key Findings

### Technical Issues Drive Volume

Technical issues represented 29.6% of all support requests.

### Priority System Works Effectively

Critical tickets were resolved 3.7x faster than low-priority tickets.

### Top Performing Agent

Ben Carter achieved the fastest average resolution time at 37.4 hours.

### Customer Satisfaction

No significant relationship was found between resolution time and customer satisfaction.

---

## Business Recommendations

1. Automate recurring technical support requests.
2. Review Billing and Account-related workflows.
3. Replicate best practices from top-performing agents.
4. Investigate additional factors influencing customer satisfaction.
